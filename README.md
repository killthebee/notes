Приложение с заметками, реализованно хранение заметок, СRUD, добавление картинок, а так же включать или менять для выделенного стандартный, жирный или курсивный шрифт.

## Requirements


```
- iOs 16
- xCode 14
```

## Запуск
Запустите симулятор в xCode
